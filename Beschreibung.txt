Eine einfache Java Anwendung (programmiert in Java 8
auf IDE VisualCode).
*************
Beschreibung:
*************
Das Programm läuft auf Windows und Linux Rechner und
verwaltet 2 Listen
=> eine Liste mit "Sherlock Holmes" Darstellern (z.B. Inspektor Lestrade, der Gute)
=> die andere Liste hält Daten von Sprechern (zum Beispiel
eines Hörbuchs) welche der entsprechenden Rolle aus Liste
1 zugeordnet sind.

Man kann Liste 1 mit einem Ausfüllformular neue Einträge hinzugeben- und aus beiden Listen EInträge löschen!!

**********************************
Das PROGRAMM lässt sich auf jedem beliebigen Browser 
öffnen. (Voraussetzung:
Java 8 installiert , Maven 3 installiert )
**********************************
